* **Difficulty:** Easy

## Walkthrough

1. Start the room's AttackBox or connect to the THM Room network via OpenVPN
2. Call `http://<machine IP address>` in your web browser
3. Look at the source code of the website in order to retrieve the guest account credentials
4. Login as user `guest`
5. Change URL's `user` parameter from `guest` to `admin`
6. Collect flag and finish room! :)
